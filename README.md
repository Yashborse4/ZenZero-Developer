# Cars0 – Used Car Marketplace Website

A modern, responsive React website for **Cars0**, a used car marketplace that connects buyers and sellers. Built with Vite, React Router, and a senior-level dark marketplace UI.

## 🌐 Live Site

[**View on GitHub Pages**](https://yashborse4.github.io/ZenZero-Developer/)

## ✨ Features

- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Dark marketplace theme (graphite + emerald + amber)
- 📄 Multi-page app with routing
- ✨ Smooth animations with Framer Motion
- 🛡️ Play Store–compliant Privacy Policy
- 📞 Contact form and account deletion flow
- 🧩 Structured CSS utilities (no Tailwind dependency)

## 📦 Pages

| Page | Path | Purpose |
|------|------|---------|
| Home | `/` | Hero, features, and marketplace positioning |
| Privacy Policy | `/privacy` | Play Store–compliant privacy policy |
| Terms of Service | `/terms` | Terms and conditions |
| Account Deletion | `/delete-account` | Account deletion request flow |
| Contact | `/contact` | Contact form and FAQs |

## 🛠️ Tech Stack

- **Framework**: React 18 + Vite
- **Routing**: `react-router-dom`
- **Animations**: `framer-motion`
- **Icons**: `lucide-react`
- **Styling**: Custom CSS with design tokens + utility layer
- **Deployment**: GitHub Pages

## 🚀 Local Development

1. Clone the repo
   ```bash
   git clone https://github.com/Yashborse4/ZenZero-Developer.git
   cd ZenZero-Developer
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start dev server
   ```bash
   npm run dev
   ```

4. Open http://localhost:5173

## 📦 Build & Deploy

### Build for production
```bash
npm run build
```

### Deploy to GitHub Pages
```bash
npm run build
git add -f dist
git commit -m "Build: Add production dist for GitHub Pages"
git push origin main
```

Then enable GitHub Pages in repo settings:
- Settings → Pages
- Source: Deploy from a branch
- Branch: main
- Folder: /(root)
